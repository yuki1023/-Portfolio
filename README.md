# hakugin
日本のゲレンデの情報＆レビューアプリです。
日本地図からゲレンデの基本情報とレビューの投稿、現在の天気とWebサイトを閲覧することができます。



## 使用技術
- 開発環境  
　   - Xcode,swift  
- DB  
　   - mBaaS,UserDefaults  
- 日本地図  
　   - MapKit  
- 天気API  
　   - OpenWeatherMap  
-json処理  
　   - Alamofire,SwiftyJSON
-タブ
　   - Tabman
-画面遷移時の非同期処理への対応
　   - PKHUD
